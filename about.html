TikZ Graph Gen

By Andrew Salls

This website is a tool for generating graphs based on the LaTeX TikZ library. LaTeX is a typesetting system primarily used for enabling easy use of symbols, diagrams, and macros, and more in documents. It is ubiquitous in the world of professional academic and scientific writing.

TikZ is a package for use within LaTeX. It is a powerful graphics library for creating all types of figures, with several other libraries that extend it for more niche visuals. More specifically, TikZ is a set of macros within the larget PGF library designed to make it easier to use without needing to understand the entire (as of writing) 1,322 page manual.

For most people using TikZ, it is too powerful for their purposes, making it hard to use. This website focuses specifically on graphs (the mathematical object from graph theory, not the tool for visualizing data), and aims to make it easier to create simple graphs within TikZ. Ideally, this makes it easier to tweak graphs as necessary.

Of course, you can use the site without knowing about or caring about LaTeX. Who am I to stop you?
